# SCIENGA SUMMITS

SCIENGA SUMMITS is a premier platform dedicated to organizing and managing global scientific conferences, summits, and meetings. This application provides a seamless experience for attendees, speakers, and organizers.

## Key Features

- **Dynamic Program Schedule**: View and manage conference sessions and timelines.
- **Abstract Submission Portal**: Easy-to-use interface for researchers to submit their work.
- **Registration System**: Streamlined process for attendee registration with multiple pricing tiers.
- **Speaker Profiles**: Detailed profiles for keynote speakers and presenters.
- **Sponsorship & Media Partners**: Dedicated sections for showcasing partners and sponsors.
- **Responsive Design**: Fully responsive interface ensuring a great experience on desktop and mobile.

## Technology Stack

- **React**: For building the user interface.
- **Vite**: For fast development and building.
- **React Router**: For client-side navigation.
- **CSS**: Custom styling for a unique and professional look.

## Getting Started

To run this project locally, follow these steps:

1.  **Prerequisites**: Ensure you have Node.js installed on your machine.

2.  **Installation**:
    Navigate to the project directory and install the dependencies:
    ```bash
    npm install
    ```

3.  **Development Server**:
    Start the local development server:
    ```bash
    npm run dev
    ```
    The application will be available at `http://localhost:5173` (or the port shown in your terminal).

4.  **Building for Production**:
    To create a production-ready build:
    ```bash
    npm run build
    ```

## Project Structure

- `src/`: Contains the source code (components, pages, assets).
- `public/`: Static assets.
- `vite.config.js`: Vite configuration file.
